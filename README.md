# Final Paycheck Deadline Calculator

A free, open-source tool to calculate when a final paycheck is due based on pay frequency and termination date.

## About

This calculator helps HR professionals, managers, and employees determine when a final paycheck should be processed after employment termination. It accounts for different pay schedules (weekly, biweekly, semimonthly, and monthly).

**Important:** This tool provides a general calculation based on standard paycheck intervals. Always verify with your state and local labor regulations, as final paycheck requirements vary by jurisdiction.

## How to Use

1. Open `paycheck-deadline-calculator.html` in any web browser
2. Select the employee's pay frequency
3. Enter the termination date
4. Click "Calculate" to see when the final paycheck is due

The tool runs entirely in your browser—no data is sent anywhere.

## Features

- Clean, mobile-friendly interface
- Instant calculation
- No installation required
- No external dependencies
- Works offline

## Customization

You can customize this tool by editing the HTML file:

- Change colors in the `<style>` section
- Modify the calculation logic in the `calculateDeadline()` function
- Add state-specific rules or notes
